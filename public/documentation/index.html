Introduction

Readykit is a user friendly admin template with some attractive Vue components. A lot of time can be saved by using the ready components. The template is built with HTML5, CSS3, Bootstrap 4, VueJS 2.6 and Laravel 9.xx. The template has almost every component required for designing and developing a web application.

  • Razorpay payment gateway integration.
